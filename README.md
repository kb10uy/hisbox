# HISBox

Various amateur radio related libraries, tools, and codes by JL1HIS.

## Binary crates

### `jelgen` - JARL eLog generator

### `qcgen` - QSL card generator

## Library crates

### `callfind` - Utility for manipulatig callsigns and grid locator

### `adif-reader` - Simple ADIF reader

## Internal crates

### `schope` - Lua scripting engine for `jelgen`/`qcgen`

### `common-qso` - Intermediate data structures
